# Mission #002 - Windows Server 2025 Virtual Machine

## Objective

Build a Windows Server 2025 virtual machine in Oracle VirtualBox to serve as the foundation for the BlackSite cybersecurity lab.

---

## Environment

Host OS:
- Windows 11

Hypervisor:
- Oracle VirtualBox 7.2

Guest OS:
- Windows Server 2025 Standard Evaluation

Resources Allocated:
- 4 vCPUs
- 8 GB RAM
- 100 GB Dynamic VDI
- EFI Enabled

---

## Tasks Completed

- Installed Oracle VirtualBox Extension Pack
- Created Windows Server virtual machine
- Attached Windows Server 2025 ISO
- Installed Windows Server successfully
- Verified Server Manager functionality
- Reviewed Local Server configuration
- Created baseline snapshot after installation

---

## Outcome

The virtual machine is fully operational and ready for Active Directory deployment in the next mission.

Snapshot Created:
Baseline - Fresh Windows Server Install

---

## Lessons Learned

- Proper VM resource allocation
- VirtualBox EFI configuration
- Windows Server installation process
- Snapshot best practices
- Importance of maintaining a clean baseline before making configuration changes
